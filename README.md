## Installation

```
# Run builder script
./builder.sh
```

```
# install dependencies
mix deps.get

# clean dependencies
mix deps.clean --all

# clean application files
mix clean

# compile project
mix compile

# Testing
mix test

# run single main method
mix run -e "TSCProject.main()"
```
